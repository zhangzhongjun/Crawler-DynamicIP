该APP是推广的一个应用，模式就是给你一个专属的推广链接，只要有人点击了这个链接，就算你推广了一个人。

典型的挂机赚钱的骗局，当你充值为会员之后，系统会封你的号。

即使是推广够了500个人，提交审核，它仍然不会让你审核通过的。

服务器会检查，如果是同一个IP来的，就算作弊。
